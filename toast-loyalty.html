How We Did It

1. Data Sources

We combined Toast POS order data (order totals, timestamps, customer phones, payment methods) with Boostly subscriber data (SMS opt-ins, phone numbers) and Toast loyalty flags (native loyalty program usage per order).

2. Customer Matching

Customers were matched across systems using phone number as the primary key. For anonymous dine-in orders (no phone captured), we used card fingerprinting (card type + last 4 digits) to link orders back to known customers.

3. Segmentation Logic

Each customer was placed into one of four segments based on:

Both = Boostly subscriber + used Toast Loyalty
Toast Only = Used Toast Loyalty, not a Boostly subscriber
Boostly Only = Boostly subscriber, never used Toast Loyalty
Neither = No program participation

4. LTV Calculation

Lifetime Value (LTV) = total revenue from all orders attributed to that customer during the analysis period. We calculated average LTV per segment to compare customer value across groups. Outliers (>100 orders) were excluded to prevent skew from caterers or business accounts.

Dataset Summary

330,384 orders across 3 locations with Toast Loyalty enabled, from January - December 2025. Card fingerprint matching recovered 68% of previously anonymous orders, enabling complete customer attribution.
